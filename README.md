# 🔐 Password Manager & Generator

A modern and responsive Password Manager built with React, Tailwind CSS, Framer Motion, and LocalStorage.

Users can generate strong passwords, check password strength, save passwords securely, search saved passwords, and manage them with an elegant UI.

## 🚀 Features

- 🔑 Generate strong random passwords
- 📏 Custom password length selection
- 🔠 Uppercase letters support
- 🔡 Lowercase letters support
- 🔢 Numbers support
- 🔣 Symbols support
- 💪 Password strength checker
- 💾 Save passwords with descriptions
- 👁️ Show / Hide passwords
- 📋 Copy password to clipboard
- ✏️ Edit password descriptions
- 🗑️ Delete passwords with confirmation popup
- 🔍 Search saved passwords
- 📱 Fully responsive design
- ⚡ Smooth animations with Framer Motion
- 🔔 Toast notifications
- 🗄️ LocalStorage persistence

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/AkashGautam-27/password-generator.git
```

### Go to Project Folder

```bash
cd password-generator
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```

---

## 💡 Usage

1. Enter a description (e.g., Gmail, GitHub, Facebook).
2. Choose password settings.
3. Click **Generate Password**.
4. Save the generated password.
5. Manage passwords using:
   - Show / Hide
   - Copy
   - Edit
   - Delete

All passwords are stored in LocalStorage and remain available after page refresh.
